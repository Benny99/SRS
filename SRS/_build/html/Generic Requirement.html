Generic Requirement¶

2.1 Features¶

The OMG application must be simple, fast, and intuitive to run, and it can be used by biologists to analyze data, process data, and sift through it. The OMG also have a submit button.It is Filtrate.After importing the data of the two gene files, we only need to click this button lightly to realize the rapid screening of the data.
